Introduction

HTML is the standard markup language for Web pages.

With HTML you can create your own Website.

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page

First code

html code could be written in many different ways

It could also be written with other programming languages also known as "manipulating the DOM" which you would learn as you progress

  • HTML can be written with html elements
  • HTML can be written with the DOM

Elements

There are many html elements for different purposes e.g "headings, paragraphs, images e.t.c..."

the different elements are:

  • h(number) tag
  • p tag
  • img tag
  • a tag
  • div tag

Images

img tags are used to put images on the web page

images need to be styled to use properly

  • img tags use src attributes as the link for the picture
  • images use alt attributes as secondary text if picture does not load
  • images use loading attributes for loading the page even if the users internet is slow

Links

Links are used to navigate to different pages or links on page

links also have different attributes like:

  • href - Which is used to put the navigation link
  • target - Which is used to open the link in new tab

More

Form are us to place labels, inputs, and buttons

There are many elements placed in form elements like:

  • inputs
  • labels
  • button
